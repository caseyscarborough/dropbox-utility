# Dropbox Utility

This application is a simple command-line utility for interacting with Dropbox.

## Installation

Add this line to your application's Gemfile:

    gem 'dropbox_utility'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dropbox_utility
